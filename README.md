# Probabilistic Robotics Algorithm

## Requirements
- Python 3.5.0+
- numpy 1.15.0+


## Installation

### pyenv
```bash
$ pyenv install 3.5.0
$ pip3 install numpy
```

### Anaconda
```bash
$ conda create -n robotics python=3.5 numpy
```

You can install requirements by `requirements.txt`
```bash
$ pip3 install -r requirements.txt
```
